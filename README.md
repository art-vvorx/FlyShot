# FlyShot
Игра в жанре шутера сверху вниз, тайм-киллера. Создана на движке Godot 3.5 для платформы Android. Кстати, рисунок тоже мой собственный. В общей сложности на проект ушло 1-2 недели. Пока игра находится в начальном состоянии, звукового оформления нет, механика довольно простая, босс пока только один, скинов нет, в общем, есть простор для развития.
